<p>※본 내용은 열혈 TCP/IP 내용을 참고 하였으며, 리눅스 환경 기반으로 테스트 하였습니다.</p>
<h2 id="네트워크-프로그래밍">네트워크 프로그래밍</h2>
<p> -&gt; 네트워크로 연결되어있는 두 컴퓨터가 데이터를 주고 받을 수 있도록 하는 것</p>
<ul>
<li>물리적 연결[유선] : 네트워크 카드 &amp; 케이블 이용</li>
<li>논리적 연결[무선] : 소켓 이용</li>
</ul>
<blockquote>
<p>소켓: 네트워크상에서 통신하는 두 프로그램 사이의 양방향 통신의 종착점(End-point)이자, 운영체제가 제공하는 추상화된 인터페이스(API)이다.</p>
</blockquote>
<p>(서버: Server) &lt;----------------------&gt; (클라이언트: Client)</p>
<ul>
<li>서버: 연결요청을 수락하는 기능 (주요 단계: Bind -&gt; Listen -&gt; Accept)</li>
<li>클라이언트: 연결 요청을 거는 기능 (주요 단계: Connect)</li>
</ul>
<hr />
<p>대표적인 소켓 2가지: TCP, UDP
이중에서 처음에 다뤄볼 소켓은 <code>TCP</code>이다.</p>
<h2 id="tcp">TCP</h2>
<blockquote>
<p>TCP(Transmission Control Protocl): 전송 제어 프로토콜이라고 부르며, 전송 계층에서 <strong>연결 지향적이고 신뢰성 있는 데이터 전송</strong>을 보장하는 프로토콜이다. </p>
</blockquote>

프로토콜: 통신 규칙, 규약.</code></pre>

#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
#include &lt;string.h&gt;
#include &lt;unistd.h&gt;
#include &lt;arpa/inet.h&gt;
#include &lt;sys/socket.h&gt;

void error_handling(char *message);

int main(int argc, char *argv[]) {
    int sock;
    struct sockaddr_in serv_addr;
    char message[30];
    int str_len;

    if (argc != 3) exit(1);

    /* -----------------------------------------------------------
     * [STEP 1] 소켓 생성 (Socket Creation)
     * ----------------------------------------------------------- */
    sock = socket(PF_INET, SOCK_STREAM, 0);

    /* -----------------------------------------------------------
     * [STEP 2] 대상 서버 주소 설정 (Server Address Setting)
     * ----------------------------------------------------------- */
    memset(&amp;serv_addr, 0, sizeof(serv_addr));
    serv_addr.sin_family = AF_INET;
    serv_addr.sin_addr.s_addr = inet_addr(argv[1]);
    serv_addr.sin_port = htons(atoi(argv[2]));

    /* -----------------------------------------------------------
     * [STEP 3] 연결 요청 (Connect)
     * ----------------------------------------------------------- */
    if(connect(sock, (struct sockaddr*)&amp;serv_addr, sizeof(serv_addr)) == -1)
        error_handling(&quot;connect() error!&quot;);

    /* -----------------------------------------------------------
     * [STEP 4] 데이터 수신 (Read)
     * ----------------------------------------------------------- */
    str_len = read(sock, message, sizeof(message) - 1);
    if(str_len==-1)
        error_handling(&quot;read() error!&quot;);
    /* -----------------------------------------------------------
     * [STEP 5] 소켓 폐쇄 (Close)
     * ----------------------------------------------------------- */
    printf(&quot;Message from server : %s\n&quot;, message);
    close(sock);

    return 0;
}

    fputc('\n', stderr);
