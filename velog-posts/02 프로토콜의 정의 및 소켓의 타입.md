<h2 id="소켓-생성">소켓 생성</h2>
<p>앞 장에서 보았듯이 소켓의 생성은 다음과 같다</p>
<pre><code class="language-c">#include &lt;sys/socket.h&gt;

int socket(int domain, int type, int protocol) 

// 위의 프로토콜 체계(PF)는 domain 자리에 들어간다</code></pre>
<ul>
<li>domain: 소켓이 사용할 프로토콜의 체계 정보 전달</li>
<li>type 소켓의 데이터 전송 방식에 대한 정보 전달</li>
<li>protocol 두 컴퓨터간 통신에 사용되는 프로토콜 정보 전달</li>
</ul>
<h2 id="소켓-타입">소켓 타입</h2>
<h3 id="1-연결-지향형-소켓sock_streamtcp">1. 연결 지향형 소켓(SOCK_STREAM)(TCP)</h3>
<p>[특징]</p>
<ul>
<li>데이터의 소멸이 없다(Reliabliity): 중간에 패킷(데이터)가 소멸되지 않고 목적지로 전송된다</li>
<li>전송 순서가 유지된다(Ordering): 1, 2, 3으로 전송 -&gt; 1, 2, 3 순서로 읽게된다</li>
<li>데이터의 경계가 없다(No Boundary): <strong>핵심포인트</strong></li>
</ul>
<p>예)
<img alt="" src="https://velog.velcdn.com/images/parang12/post/8fb7fd0b-e455-42a0-90cf-d6af9c82aaab/image.jpg" /></p>
<p>소켓에는 버퍼(임시저장소)가 존재함 &lt;송신버퍼, 수신버퍼&gt;</p>
<pre><code class="language-text">1. write()함수로 데이터를 송신버퍼에 넣음 - 사용자
2. 송신버퍼에 쌓인 데이터를 패킷 형태로 만든뒤 수신자에게 보냄 - OS
3. 받은 데이터가 수신버퍼에 쌓임 read()함수를 통해서 가져감 - 사용자</code></pre>
<p>if 소켓에 존재하는 수신버퍼가 꽉차면 데이터는?
-&gt; 수신 버퍼가 꽉차면, 수신 측 OS가 송신측 OS에게 꽉찼다고 ACK패킷을 보냄 -&gt; 그 뒤에 송신 버퍼까지 꽉차면 wirte()함수도 멈춤 (blocking)</p>
<p>tip) &quot;리눅스는 모든것이 파일이다&quot;라는 것 때문에 소켓도 read()와 write()함수를 이용해서 데이터를 주고 받는다</p>
<h3 id="2-비-연결-지향형-소켓sock_dgramudp">2. 비 연결 지향형 소켓(SOCK_DGRAM)(UDP)</h3>
<p>[특징]</p>
<ul>
<li>전송된 순서에 상관없이 가장 빠른 전송을 지향한다.</li>
<li>전송된 데이터는 손실의 우려가 있고, 파손의 우려가 있다.</li>
<li>전송된는 데이터의 경계가 존재한다.</li>
<li>한번에 전송할 수 있는 데이터의 크기가 제한된다.</li>
</ul>
<p><img alt="" src="https://velog.velcdn.com/images/parang12/post/3da0f902-ad08-47cc-83d7-10e19bffe92d/image.jpg" /></p>
<pre><code class="language-text">1. sendto() 함수를 호출하여 데이터를 송신버퍼에 넣음 - 사용자
2. 송신버퍼에 데이터를 즉각적으로 네트워크 카드 형태로 보냄 - OS
3. 수신 버퍼에 쌓인 데이터를 덩어리 recvfrom()으로 가져감 - 사용자</code></pre>
<p>if 소켓에 존재하는 수신버퍼가 꽉차면 데이터는?
-&gt; 수신 버퍼가 꽉차면 그냥 받은 데이터를 지워버림</p>
<h2 id="tcp-예시">TCP 예시</h2>
<pre><code class="language-c">#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
#include &lt;string.h&gt;
#include &lt;unistd.h&gt; 
#include &lt;arpa/inet.h&gt;
#include &lt;sys/socket.h&gt;

void error_handling(char *message);

int main(int argc, char **argv) {
    int sock; 
    struct sockaddr_in serv_addr; 
    char message[30]; 
    int str_len = 0;
    int idx = 0, read_len = 0;

    if (argc != 3) { 
        printf(&quot;Usage: %s &lt;IP&gt; &lt;port&gt;\n&quot;, argv[0]);
        exit(1);
    }

    /* -----------------------------------------------------------
     * [STEP 1] 소켓 생성 (Socket Creation)
     * : 운영체제로부터 네트워크 통신에 필요한 자원(소켓)을 할당받습니다.
     * ----------------------------------------------------------- */
    sock = socket(PF_INET, SOCK_STREAM, 0); // TCP 소켓, PF_INET = IPv4 프로토콜
    if (sock == -1) 
        error_handling(&quot;socket() error&quot;);

    /* -----------------------------------------------------------
     * [STEP 2] 서버 주소 설정 (Server Address Setting)
     * : 접속할 서버의 IP 주소와 포트 번호를 구조체에 기록합니다.
     * ----------------------------------------------------------- */
    memset(&amp;serv_addr, 0, sizeof(serv_addr));
    serv_addr.sin_family = AF_INET;
    serv_addr.sin_addr.s_addr = inet_addr(argv[1]); 
    serv_addr.sin_port = htons(atoi(argv[2]));     

    /* -----------------------------------------------------------
     * [STEP 3] 서버에 연결 요청 (Connect)
     * : 이제 클라이언트가 서버에 접속을 시도합니다.
     * ----------------------------------------------------------- */
    if (connect(sock, (struct sockaddr*)&amp;serv_addr, sizeof(serv_addr)) == -1)
        error_handling(&quot;connect() error&quot;);

    /* -----------------------------------------------------------
     * [STEP 4] 데이터 송수신 (Data Transmission)
     * : 서버로부터 메시지를 받아 화면에 출력합니다.
     * ----------------------------------------------------------- */

    while ((read_len = read(sock, message + idx, sizeof(message) - 1 - idx)) &gt; 0) { // 서버로부터 데이터를 읽어옵니다.
        if (read_len == -1){// 읽기 오류 발생 시 에러 처리
            error_handling(&quot;read() error&quot;);
            break;
        }
        str_len += read_len; // 총 읽은 바이트 수 누적

    }
    /* -----------------------------------------------------------
     * [STEP 5] 소켓 닫기 (Socket Close)
     * : 더 이상 사용하지 않는 소켓을 닫습니다.
     * ----------------------------------------------------------- */
    printf(&quot;Message from server: %s&quot;, message);
    printf(&quot;Function read call count: %d\n&quot;, str_len);
    close(sock); 
    return 0;
}

void error_handling(char *message) {
    fputs(message, stderr);
    fputc('\n', stderr);
    exit(1);
}</code></pre>
<p>이 코드를 통해서 TCP의 데이터의 경계가 없다는 것을 알 수 있음</p>
